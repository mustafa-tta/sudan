# sudan
يهدف هذا المشروع إلى إنشاء منصة إلكترونية تساعد المنظمات الإنسانية والخيرية على إدارة وتوزيع المساعدات للولايات المتأثرة بالنزوح في السودان بطريقة منظمة وفعالة. يتيح النظام تسجيل طلبات المساعدات من مختلف الولايات، مع تحديد نوع الاحتياج والكمية المطلوبة ودرجة الأولوية، مما يسهل على المنظمة متابعة الطلبات واتخاذ قرارات التوزيع بصورة عادلة وسريعة.
